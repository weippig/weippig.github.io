## Getting Start
1. install Ruby Gems
``` shell
bundle install
```
2. run jekyll server(default: localhost:4000)
``` shell
bundle exec jekyll serve
```
3. If you add some new posts including new tags
``` shell
python3 tag_generator.py
```
